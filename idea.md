# LibraLearn  
### Smart College Library Management System with Recommendation Engine

## Project Overview

LibraLearn is a smart college library management system designed to manage physical books efficiently while enhancing student engagement through personalized book recommendations.

Unlike traditional library systems that only handle inventory and borrowing operations, LibraLearn introduces a recommendation layer that analyzes student borrowing behavior to suggest relevant books. The goal is to combine operational efficiency with intelligent user experience.

---

## Problem Statement

Traditional college library systems focus only on:

- Book inventory management  
- Borrow and return tracking  
- Basic record keeping  

They do not assist students in discovering books aligned with their interests.

LibraLearn solves this by integrating a recommendation engine that suggests books based on borrowing history and popularity trends.

---

## Core Features

### Student
- Register and login
- Browse and search books
- Borrow and return books
- View borrowing history
- Rate and review books
- Receive personalized recommendations

### Librarian
- Add, update, and delete books
- Manage inventory
- View borrowing records
- Track overdue books
- View most borrowed books

---

## Key Highlights

- Layered backend architecture (Controller → Service → Repository)
- Role-based authentication and authorization
- Fine calculation system for overdue books
- Strategy Pattern for recommendation logic
- Clean and modular system design
- Scalable structure for future ML-based recommendations
